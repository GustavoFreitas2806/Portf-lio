Competências Desenvolvidas

Framework & Page Objects

Criação de projetos do zero utilizando o padrão Page Object para organização. Desenvolvimento de uma DSL (Domain Specific Language) própria para tornar os testes mais enxutos, legíveis e fáceis de manter.

Sincronismo & Elementos

Estratégias avançadas para localização de elementos (XPaths robustos) e interação com componentes complexos como Alerts, Frames e PopUps.

Execução Grid & Cloud

Experiência na execução de testes em paralelo, sequencial ou remotamente utilizando Selenium GRID e serviços de nuvem (Saucelabs), cobrindo múltiplos browsers (Firefox, Chrome, IE).

Testes de API & Exceptions

Realização de testes de consumo de APIs, implementando tratamento rigoroso de erros com Exceptions personalizadas para garantir que a falha seja identificada e tratada corretamente, aumentando a resiliência da automação.

Tipos de Testes Realizados

Testes de Regressão

Execução de baterias de testes automatizados para garantir que novas alterações no código (deploys) não quebraram funcionalidades que já estavam funcionando no sistema legado.

Testes de Caixa Preta

Validação funcional focada nas entradas e saídas (Inputs/Outputs), testando a aplicação sob a perspectiva do usuário final, sem considerar a estrutura interna do código.

Testes Unitários

Utilização do framework JUnit para testar as menores unidades de código (métodos e classes) de forma isolada, garantindo a lógica de programação antes da integração.

Testes Funcionais

Verificação dos requisitos de negócio e histórias de usuário, simulando a interação real (cliques, preenchimentos e navegação) para assegurar que o software faz o que deve fazer.
